# 🚀 GitHub Clone (MERN Stack)

A full-stack GitHub-inspired web application where users can create repositories, explore projects, and manage their own code workspace.

---

## ✨ Features

- 🔐 User Authentication (Signup / Login)
- 📦 Create & Manage Repositories
- 🔍 Search Repositories
- 👤 User Dashboard with personal repos
- 🌍 Explore Suggested Repositories
- ⚡ REST API with Express & MongoDB

---

## 🛠️ Tech Stack

### Frontend
- React.js
- CSS (GitHub-style UI)
- Axios / Fetch

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
